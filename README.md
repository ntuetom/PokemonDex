# PokemonDex

## First
`pod install`

## Feature:
- MVVM-C
- API request
- User favorite pokemon save
- Dynamic loading
- Pokemon EvoChain 
- Dependency injection
- DataBase to enhance pokemon viewer experience
- Some UI Test

## Dependency:
- 'SnapKit'
- 'Moya'
- 'Alamofire'
- 'RxSwift'
- 'RxDataSources'
- 'Kingfisher'
- 'SQLite.swift'
